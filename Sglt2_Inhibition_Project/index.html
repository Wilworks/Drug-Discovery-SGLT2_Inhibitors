SHAP Analysis & Explainability

Feature Importance Analysis

SHAP (SHapley Additive exPlanations) values provide insights into how each molecular fingerprint bit contributes to the model's predictions. The analysis reveals the most important structural features that determine SGLT2 inhibitory activity.

SHAP plots and detailed analysis available in the interactive prediction tool.

About the Project

Methodology

  • Data Curation: Molecular data from PubChem database
  • Data Preprocessing: Undersampling for class balance
  • Feature Representation: ECFP4 fingerprints (2048 bits)
  • Models: Random Forest, Gradient Boosting, XGBoost, CatBoost, SVM
  • Evaluation: 80/20 stratified split with comprehensive metrics
  • Explainability: SHAP for model interpretability

Key Results

The XGBoost model achieved the best performance with an accuracy of 0.8846, precision 0.8400, recall 0.9130, F1-score 0.8750, and ROC-AUC 0.9799, outperforming other classifiers.
